# OSDev
This repository contains my own notes and implementations following the [OSDev](https://wiki.osdev.org/Expanded_Main_Page) tutorials and articles.

## Origin
This project comes from my natural interest in low-level programming.
Starting to being interested in UEFI programming after doing a [simple bootloader](https://github.com/Keviannn/simple_bootloader) using BIOS I ended up again in the OSDev webpage (they say all roads lead to Rome), so I said "let's do things how they are supposed to" and here we are.

## This repository
The purpose of this repository is for me to have a way to track my progress, structure my project and have a backup. I would't recommend to use it as a guide as is based on a real wiki that will for sure have less mistakes than my own implementations.

## Purpose
The purpose of this project is for me to go step by step digging more into low level programming and expanding my knowledge about operating systems, how they are structured and make programs with more awareness about the hardware and the code that carries all the modern software.

Making a super complex and usable OS is not my main goal, as I am still an undergraduate and don't have the necessary knowledge, but I still want to learn about UEFI, bootloaders, kernels and of course some kind of attempts will appear here.

## Structure
For now the structure will be a directory for every tutorial or piece of code with its own README for me to add notes and review in the future, probably with links to the OSDev webpage.

If the project ends up growing over this approach, I will desing a more compact and navigable structure.
